# aws-technical-essentials-tf

Terraform configuration for Employee Directory Application example in AWS Technical Essentials

## Quick Start

```bash
cd assets/
terraform init
terraform plan
terraform apply
```

## Variables

| Variable | Default value |
| --- | --- |
| `region` | `ap-east-1` |
| `instance_type` | `t3.micro` |

## Supported regions

`ap-east-1`, `us-west-2`
